# Instacart 재구매 예측 AI 모델 개발 (캡스톤)

## 시작하기 전 가이드

1. 저장소를 클론(Clone)합니다.
2. 터미널에서 라이브러리를 설치합니다: `pip install -r requirements.txt`
3. 공유드린 노션에서 '문서허브>캐글 API 키 발급' 문서를 확인 후 터미널에서 데이터를 다운로드받아 `data/` 폴더에 넣어주세요. (데이터는 .gitignore에 의해 공유되지 않습니다.)

## 📂 폴더 구조

- `data/`: 원본 데이터셋 (.csv)
- `notebooks/`: 팀원별 분석 주피터 노트북 (.ipynb)
- `src/`: 공통 함수 및 모델 소스 코드
